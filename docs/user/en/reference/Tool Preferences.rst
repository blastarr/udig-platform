Available Preferences
~~~~~~~~~~~~~~~~~~~~~

-  default feature attribute - name of feature attribute to be displayed in popup when multiple features are detected (case insensitive)

-  selection radius - controls the snap radius (in pixel) during single feature selection like commands (edit, select etc.). During mouse click
   on the map all features found within this radius will be appear in an popup (using the default feature attribute mentioned above) for the user 
   to select from (see figure below)
